API Referenz ​

Die ScoreHub API ist eine RESTful API, die auf Azure Functions basiert.

Basis-URL ​

UmgebungURL
Staginghttps://scorehub-staging-api.azurewebsites.net/api
Productionhttps://scorehub-production-api.azurewebsites.net/api
Lokalhttp://localhost:7071/api

Authentifizierung ​

Alle Endpoints (ausser /health) erfordern einen Bearer Token:

http
Authorization: Bearer <access_token>

Der Token wird von Azure AD B2C ausgestellt.

Endpoints ​

Health Check ​

http
GET /health

Keine Authentifizierung erforderlich.

Response:

json
{
  "status": "healthy",
  "timestamp": "2024-01-15T10:30:00Z",
  "version": "1.0.0"
}

Clubs ​

MethodEndpointBeschreibung
GET/clubsAlle Vereine abrufen
GET/clubs/{id}Einzelnen Verein abrufen
POST/clubsNeuen Verein erstellen
PUT/clubs/{id}Verein aktualisieren
DELETE/clubs/{id}Verein löschen

Competitions ​

MethodEndpointBeschreibung
GET/competitionsAlle Wettkämpfe abrufen
GET/competitions/{id}Einzelnen Wettkampf abrufen
POST/competitionsNeuen Wettkampf erstellen
PUT/competitions/{id}Wettkampf aktualisieren
DELETE/competitions/{id}Wettkampf löschen

Results ​

MethodEndpointBeschreibung
GET/competitions/{id}/resultsResultate eines Wettkampfs
POST/resultsNeues Resultat erfassen
PUT/results/{id}Resultat bearbeiten
DELETE/results/{id}Resultat löschen

Rankings ​

MethodEndpointBeschreibung
GET/competitions/{id}/rankingRangliste generieren
GET/competitions/{id}/ranking/pdfRangliste als PDF

Error Responses ​

json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Name ist ein Pflichtfeld",
    "details": [
      { "field": "name", "message": "Pflichtfeld" }
    ]
  }
}

HTTP Status Codes ​

CodeBeschreibung
200Erfolg
201Erstellt
400Validierungsfehler
401Nicht authentifiziert
403Keine Berechtigung
404Nicht gefunden
500Server-Fehler

Pagination ​

Listen-Endpoints unterstützen Pagination:

http
GET /competitions?page=1&pageSize=20

Response:

json
{
  "data": [...],
  "pagination": {
    "page": 1,
    "pageSize": 20,
    "totalPages": 5,
    "totalCount": 95
  }
}

Filtering ​

http
GET /competitions?status=open&year=2024
GET /results?shooterId=xxx&competitionId=yyy

Rate Limiting ​

  • 100 Requests pro Minute pro User
  • 429 Too Many Requests bei Überschreitung

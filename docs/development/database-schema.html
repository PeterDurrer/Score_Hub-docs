Datenbankschema ​

Diese Seite beschreibt das vorgeschlagene neue Datenbankschema von ScoreHub, einschließlich aller Tabellen, Felder, Datentypen und Beziehungen.

Übersicht ​

ScoreHub verwendet Azure SQL Database mit folgenden Hauptbereichen:

  • Benutzer & Authentifizierung: users, roles, userRoles
  • Vereine & Organisationen: shootingRangeManufacturers, clubs, clubLocations, clubUsers
  • Ausrüstung & Kategorien: equipments, ageCategories, licenseCategories, disciplines
  • Schützen: shooters
  • Wettkämpfe: competitions, competitionAwards, competitionCategories, competitionRounds, competitionRoundAwardMatrix, competitionRankings, competitionshooters, competitionGroups, competitionGroupShooters, competitionRoundResults

Setup Übersicht ​

Diese Anleitung führt Sie durch die komplette Einrichtung von ScoreHub für Ihre Organisation.

Setup Schritte ​

1. Voraussetzungen ​

Installieren Sie alle benötigten Tools und erstellen Sie ein Azure Konto.

2. Azure Infrastruktur ​

Deployen Sie die Azure Ressourcen mit Bicep Templates.

3. Azure AD B2C ​

Konfigurieren Sie die Authentifizierung mit Azure AD B2C.

4. Datenbank ​

Initialisieren Sie die SQL Datenbank mit dem Schema.

5. GitHub Secrets ​

Konfigurieren Sie die CI/CD Pipeline Secrets.
